Set the default values in the EsBeaconService. All parameters are the same as they are for the attributes on the es-beacon-element.

Here is a list of all setter-functions and their default values.
  • beacon.setDefaultColor("hsl(235, 70%, 50%)")
  • beacon.setDefaultWidth("10px")
  • beacon.setDefaultSize("12px")
  • beacon.setDefaultFrequency("2000ms")
  • beacon.setDefaultSynchronisation(false)
  • beacon.setDefaultTop("-12px")
  • beacon.setDefaultBottom("")
  • beacon.setDefaultLeft("-12px")
  • beacon.setDefaultRight("")
