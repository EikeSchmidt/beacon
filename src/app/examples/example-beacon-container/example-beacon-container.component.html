The beacons position will be relative to the next parent element with a css position of "relative", "absolute" or "fixed".

For better readability, you can use the esBeaconContainer-directive which applies the "relative"-position-attribute.
You can also give parameters to the attribute to change the attribute.

  • esBeaconContainer="absolute" will apply positon:absolute
  • esBeaconContainer="fixed" will apply positon:fixed
  • [esBeaconContainer]="false", [esBeaconContainer]="null" and esBeaconContainer="static" will apply positon:static, hence disable this element as a beacon-container.
  • everything else will apply position:relative
